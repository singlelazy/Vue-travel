初始化项目
    vue init webpack 项目名
安装依赖
    npm install
    安装 vue 路由模块,和网络请求模块 
        npm install vue-router axios --save
        在src下的main.js里注册并使用
            import axios from 'axios'
            Vue.prototype.$ajax= axios
            使用时
                this.$ajax.get('').then((res)=>{}).catch((res)=>{})
